For Loops
Go through a collection one item at a time

The Big Idea

A for loop says: "Go through this collection, and for each item, do something." You write the action once — Python repeats it for every item.

# Read out loud: "FOR each name IN names, print it"
names = ["Alice", "Bob", "Charlie"]

for name in names:
    print(name)

# Alice
# Bob
# Charlie
for … in …
works on lists, strings, range()
indented block repeats
3 / 18

For Loop — What Actually Happens 🔄

THE CODE
fruits = ["apple", "mango", "grape"]

for fruit in fruits:
    print(f"I like {fruit}")
fruit is the loop variable. Python sets it to the next item automatically each round. You can name it anything!
STEP-BY-STEP TRACE
Round 1 fruit = "apple" → prints I like apple
Round 2 fruit = "mango" → prints I like mango
Round 3 fruit = "grape" → prints I like grape
List empty → loop ends
INDENTATION RULE:
✅ Indented → inside loop, repeats each round
❌ Not indented → outside loop, runs once after

For Loop — range() and Strings 🔄

COUNTING WITH range()
# range(5) → 0, 1, 2, 3, 4
for i in range(5):
    print(i)

# range(1, 6) → 1, 2, 3, 4, 5
for i in range(1, 6):
    print(i)

# range(1, 10, 2) → 1, 3, 5, 7, 9
for i in range(1, 10, 2):
    print(i)
LOOPING OVER A STRING
# A string is just a list of letters!
word = "Python"

for letter in word:
    print(letter)

# P
# y
# t
# h
# o
# n
range(start, stop, step) — stop is not included. range(1,6) gives 1–5, not 1–6.

For Loop — 3 Patterns You Must Know 🔄

ACCUMULATOR
# Start with 0, add each item
scores = [80, 90, 70]
total = 0

for s in scores:
    total = total + s

print(total)   # 240
Start the "bucket" at 0. Add each item every round.
FILTER (if inside loop)
# Only act on some items
nums = [1,2,3,4,5,6]

for n in nums:
    if n % 2 == 0:  # even?
        print(n)

# 2  4  6
Use if inside the loop to skip items you don't need.
BUILD A LIST
# Collect results into a new list
names = ["alice", "bob"]
upper = []

for n in names:
    upper.append(n.upper())

print(upper)
# ['ALICE', 'BOB']
Start with an empty list []. Append items each round.
💡 These 3 patterns cover 90% of what you'll do with loops. Master them and everything else is easy.

While Loops
Keep going until a condition turns False

When You Don't Know How Many Times

Use while when you don't know upfront how many repetitions you need — you just know the condition to keep going.

count = 5

while count > 0:     # keep going AS LONG AS count > 0
    print(count)
    count -= 1         # ← ALWAYS change something!

print("Go!")
while condition:
always change something inside!
stops when condition = False
7 / 18

While Loop — Step by Step 🔁

HOW IT WORKS
① Check the condition
② True → run the block → go back to ①
③ False → exit the loop
count=5 → 5>0? YES → print 5 → count=4
count=4 → 4>0? YES → print 4 → count=3
count=3 → 3>0? YES → print 3 → count=2
count=2 → 2>0? YES → print 2 → count=1
count=1 → 1>0? YES → print 1 → count=0
count=0 → 0>0? NO → EXIT → print "Go!"
THE INFINITE LOOP — BIGGEST MISTAKE
# ❌ BAD — runs forever!
count = 5
while count > 0:
    print(count)
    # forgot to change count!
    # count is always 5 → always True → forever!
# ✅ GOOD — changes count every round
count = 5
while count > 0:
    print(count)
    count -= 1   # ← this makes it stop!
⚠️ Every while loop MUST change something that eventually makes the condition False. Press Ctrl+C to stop an infinite loop.

For vs While + break & continue 🔁

FOR vs WHILE — WHEN TO USE WHICH
for loop
Use when you know the collection or count upfront
for name in names: ...
while loop
Use when you keep going until something changes
while user != "quit": ...
break AND continue
# break — EXIT the loop immediately
for n in [1,2,3,4,5]:
    if n == 3:
        break         # stops the whole loop
    print(n)           # prints 1, 2  (stops before 3)

# continue — SKIP this round, go to next
for n in [1,2,3,4,5]:
    if n == 3:
        continue      # skip 3, keep going
    print(n)           # prints 1, 2, 4, 5

Nested Loops
A loop living inside another loop

Every Combination, Automatically

A nested loop is just a loop inside another loop. For every 1 step of the outer loop, the inner loop runs all its steps.

colours = ["Red", "Blue"]
sizes   = ["S", "M", "L"]

for colour in colours:       # outer — runs 2 times
    for size in sizes:         # inner — runs 3 times PER colour
        print(colour, size)   # 2 × 3 = 6 total prints
outer × inner = total
inner always completes fully
use extra indentation

Nested Loops — Watch It Run 🔲

THE CODE
for i in range(1, 4):   # i: 1, 2, 3
    for j in range(1, 4): # j: 1, 2, 3
        print(f"i={i} j={j}")
How many lines?
outer runs 3 times (i=1,2,3)
inner runs 3 times per outer
Total: 3 × 3 = 9 lines
FULL TRACE
── i = 1 ──
  j=1 → i=1, j=1
  j=2 → i=1, j=2
  j=3 → i=1, j=3
── i = 2 ──
  j=1 → i=2, j=1
  j=2 → i=2, j=2
  j=3 → i=2, j=3
── i = 3 ──
  j=1 → i=3, j=1   … j=2 … j=3

Nested Loops — Real Examples 🔲

MULTIPLICATION TABLE
for row in range(1, 4):
    for col in range(1, 4):
        print(row * col, end="  ")
    print()   # new line after each row

# 1  2  3
# 2  4  6
# 3  6  9
STAIRCASE PATTERN
for row in range(1, 6):
    for col in range(row):  # inner uses outer!
        print("#", end="")
    print()

# #
# ##
# ###
# ####
# #####
The inner loop can use the outer loop's variable — that's how the staircase grows!

Functions
Write once, use anywhere

What Is a Function?

A function is a named block of code that you can run whenever you need it. Think of it as a recipe: write it once, cook it any time.

# DEFINE the function (write the recipe)
def greet(name):
    print(f"Hello, {name}!")

# CALL the function (cook the dish)
greet("Alice")   # Hello, Alice!
greet("Bob")     # Hello, Bob!
greet("Charlie") # Hello, Charlie!
def name(params):
indented body
call it with name(args)

Functions — The Four Parts 🧩

def add_numbers(a, b):
result = a + b
return result
def
Keyword that tells Python: "I'm creating a function"
add_numbers
The name — use it to call the function later
a, b
Parameters — the inputs the function expects
return
Sends the result back to whoever called the function
USING THE FUNCTION
# Call it and SAVE the result
answer = add_numbers(3, 4)
print(answer)      # 7

# Use the result directly
print(add_numbers(10, 20))  # 30

# Use it in an expression
total = add_numbers(5, 5) * 2
print(total)       # 20
💡 def only defines the function — it doesn't run it. You must call it to run it.

print vs return + Default Parameters 🧩

print vs return — THE KEY DIFFERENCE
def square_print(n):
    print(n * n)      # just SHOWS it

def square_return(n):
    return n * n      # SENDS it back

x = square_print(5)   # x = None !
y = square_return(5)  # y = 25 ✓
print(y * 2)            # 50 — can use it!
Use return when you need to do something with the result. Use print when you just want to see it.
DEFAULT PARAMETERS
def greet(name, msg="Hello"):
    print(f"{msg}, {name}!")

greet("Alice")              # Hello, Alice!
greet("Bob", "Good morning") # Good morning, Bob!
greet("Charlie", "Hey")      # Hey, Charlie!
Default params go at the END of the parameter list
If the caller passes a value, it OVERRIDES the default

Functions + Loops — The Power Combo 🧩🔄

Define a function once, then call it inside a loop. This is how real programs are structured.

# Step 1: define the helper function
def get_grade(score):
    if score >= 90:
        return "A"
    elif score >= 75:
        return "B"
    elif score >= 60:
        return "C"
    else:
        return "F"
# Step 2: call it inside a loop
scores = [88, 95, 62, 45, 73]

for score in scores:
    grade = get_grade(score)   # ← function call!
    print(f"{score} → {grade}")

# 88 → B
# 95 → A
# 62 → C
# 45 → F
# 73 → B
💡 The function handles the logic for one score. The loop applies it to all scores. This is the foundation of clean, real-world code.

Common Mistakes — Don't Fall For These! ⚠️

❌ Wrong indentation
for n in nums:
print(n)  ← NOT indented → error!
Body must be indented with 4 spaces or a tab
❌ Forgetting the colon
for n in nums   ← no colon → SyntaxError!
while x > 0    ← no colon → SyntaxError!
def greet(name) ← no colon → SyntaxError!
❌ Infinite while loop
count = 5
while count > 0:
    print(count)
    # forgot: count -= 1
Always change the variable the condition checks!
❌ Calling a function before defining it
greet("Alice")  ← called before def!

def greet(name):
    print(name)
Always define the function first, then call it
